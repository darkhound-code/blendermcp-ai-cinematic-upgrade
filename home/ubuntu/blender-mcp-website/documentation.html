Getting Started

Installation

Follow these steps to install BlenderMCP Ultimate Cinematic Upgrade:

Requirements

  • Blender 2.83 or newer
  • Python 3.7 or newer
  • Internet connection for asset downloading

Installation Steps

  1. Download the latest release from the Download page or GitHub Releases.
  2. Open Blender and go to Edit > Preferences > Add-ons.
  3. Click "Install..." and select the downloaded ZIP file.
  4. Enable the addon by checking the box next to "AI: BlenderMCP Ultimate".

The Blender Add-ons preferences panel showing BlenderMCP Ultimate installed and enabled.

Note

If you encounter any issues during installation, please check the Troubleshooting section or contact support.

Quick Start Guide

Get started with BlenderMCP Ultimate in just a few steps:

  1. Open Blender and create a new scene.
  2. Open the BlenderMCP Ultimate panel in the sidebar (N key).
  3. Enter your API key in the settings tab (if required).
  4. Enter a text prompt describing the cinematic sequence you want to create.
  5. Click "Generate Cinematic Sequence".
  6. Wait for the AI to create your sequence.

User Interface

The BlenderMCP Ultimate panel is divided into several tabs:

Overview of the BlenderMCP Ultimate panel in Blender's sidebar.

Main Tab

The main tab provides a simple interface for creating complete cinematic sequences from text prompts. Enter your description and click "Generate" to create a sequence.

Scene Setup Tab

This tab contains tools for importing assets and creating environments. You can search for assets, apply HDRI lighting, and generate procedural environments.

Animation Tab

The animation tab provides tools for rigging characters, creating animations, and setting up cameras. You can generate animations from text descriptions or use the manual animation tools.

Physics & VFX Tab

This tab contains tools for adding physics simulations and visual effects to your scene. You can add cloth, hair, rigid body physics, and particle effects.

Rendering Tab

The rendering tab provides tools for configuring render settings and post-processing effects. You can select render engines, apply post-processing effects, and set up multi-camera rendering.

Settings Tab

The settings tab allows you to configure API keys, performance settings, and other preferences.

Command Reference

BlenderMCP Ultimate provides a command-based system for accessing all features programmatically. This section provides a complete reference for all available commands.

Command Structure

Commands in BlenderMCP Ultimate follow a hierarchical structure:

module.submodule.function

For example:

scene_setup.hdri_lighting.apply_hdri

Command Examples

Command Description Parameters
asset_management.mixamo.import_character Import a character from Mixamo character_type: Type of character to import
name: Name for the imported character
scene_setup.hdri_lighting.apply_hdri Apply an HDRI environment map hdri_name: Name of the HDRI
strength: Light strength
character_animation.keyframe_generation.generate_animation Generate animation from text character_name: Name of the character
animation_type: Type of animation
prompt: Text description
duration: Animation duration

For a complete list of commands, see the full Command Reference.
