Installation Instructions

1

Download the Addon

Download the latest version of BlenderMCP Ultimate from the links above.

2

Open Blender Preferences

Open Blender and go to Edit > Preferences > Add-ons.

3

Install the Addon

Click "Install..." and select the downloaded ZIP file.

4

Enable the Addon

Enable the addon by checking the box next to "AI: BlenderMCP Ultimate".

5

Access the Addon

Open the BlenderMCP Ultimate panel in the sidebar (N key).
